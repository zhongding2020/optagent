# 试验设计 (DOE)

## When to Use
在确定关键工艺参数后，使用本 Skill 设计试验方案。

## Steps
1. 确定试验目标（筛选因子/优化参数/验证模型）
2. 选择合适的DOE方法（全因子/部分因子/正交/响应面）
3. 确定试验次数和排列
4. 制定试验执行计划（顺序、随机化、重复）
5. 输出DOE设计矩阵

## Output Format
DOE设计包含: 方法, 因子数, 水平数, 试验次数, 设计矩阵, 执行计划
