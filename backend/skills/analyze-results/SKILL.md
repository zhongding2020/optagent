# 数据分析与因子提取

## When to Use
在收集完试验数据后，使用本 Skill 分析数据并提取关键因子。

## Steps
1. 数据预处理（标准化/归一化）
2. 因子效应分析（主效应图/Pareto图）
3. 方差分析(ANOVA)
4. 相关性分析（相关系数矩阵/热力图）
5. 交互效应分析
6. 关键因子排序和筛选
7. 输出分析结果

## Output Format
分析结果包含: factor_ranking, pareto_data, correlation_matrix, anova_results, interaction_effects
