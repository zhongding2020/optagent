# 生成验证报告

## When to Use
在完成数据分析后，使用本 Skill 生成最终优化报告。

## Steps
1. 汇总优化目标和约束条件
2. 总结DOE设计和执行情况
3. 呈现关键因子分析结果
4. 推荐最优参数组合
5. 验证和风险评估
6. 提出下一步建议
7. 输出完整验证报告

## Output Format
报告包含: summary, recommended_parameters, analysis_charts, risk_assessment, next_steps
