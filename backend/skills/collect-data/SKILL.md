# 收集试验结果

## 核心规则
- **严禁**生成或执行 Python/代码。通过对话引导用户正确收集数据。
- 完成本步骤后务必调用 `step_complete` 工具。

## Steps

### 1. 试验执行确认
- 确认DOE试验按计划执行
- 检查试验顺序是否随机化
- 确认所有测量设备已校准

### 2. 数据收集清单
- 结果变量（响应值）的测量
- 过程参数的记录（实际值 vs 设定值）
- 环境条件记录
- 异常事件记录

### 3. 数据质量检查
- 缺失值处理
- 异常值识别
- 测量系统评估（重复性和再现性）

### 4. 输出数据摘要
- 调用 `step_complete` 工具，输出格式：
```
{
  "total_samples": 样本数,
  "response_range": {"min": 最小值, "max": 最大值, "mean": 平均值, "std": 标准差},
  "missing_values": 缺失值数量,
  "outliers": 异常值数量,
  "data_quality": "数据质量评估描述"
}
```
