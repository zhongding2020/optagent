# 识别关键工艺参数

## When to Use
在明确优化目标后，使用本 Skill 识别影响工艺目标的关键参数。

## Steps
1. 列举所有可能影响目标的工艺参数
2. 筛选关键参数（基于工程经验或初步分析）
3. 确定每个参数的取值范围
4. 评估参数间的交互可能性
5. 确定可控参数和噪声参数
6. 输出关键参数清单

## Output Format
参数列表包含: 参数名称, 单位, 取值范围, 可调性, 预期影响方向
