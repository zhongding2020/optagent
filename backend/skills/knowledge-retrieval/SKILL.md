# 知识库检索

## 核心规则
- 主动使用 `query_knowledge_base` 工具检索信息，不要凭记忆回答。
- 查询时使用准确的关键词（如"注塑飞边 解决方案"、"DOE 全因子设计"等）。
- 将检索结果与当前用户问题结合，提供有依据的建议。

## Steps

### 1. 识别知识需求
- 当用户问到工艺参数、材料特性、行业标准、历史案例时，首先检索知识库
- 典型场景：
  - 工艺参数范围（注塑温度、压力标准）
  - 材料规格（收缩率、流动性）
  - DOE方法参考（正交表设计）
  - 缺陷分析与解决方案

### 2. 执行检索
- 使用 `query_knowledge_base` 工具，构造精准的查询语句
- 设置合适的 top_k 值（通常 3-5 条）
- 可根据需要多次查询不同关键词

### 3. 结果运用
- 引用知识库内容时标明信息来源
- 将知识库信息与用户具体情况结合分析
- 如果知识库没有相关信息，说明情况并提供通用建议
